# 设置许可证密钥

<Callout type="info">
  This article is compatible with FeatBit 3.0.0 and above.
</Callout>


本指南将指导您如何使用许可证密钥来设置 FeatBit。

## 获取工作区 ID

在获取许可证密钥之前，您需要知道工作区的 ID。请按照以下步骤操作：

1. 确保您的 FeatBit 正在运行，并且 FeatBit 用户界面可以通过 http://localhost:8081 访问
2. 使用您的邮箱和密码登录 FeatBit。默认设置如下：
   - 用户名：test@featbit.com
   - 密码：123456

3. 访问 http://localhost:8081/en/workspace，在 **许可证设置** 区域，点击 **复制** 按钮

   ![](./assets/workspaceid.png)

## 获取免费试用许可证密钥

要评估所有 FeatBit 功能，请从[FeatBit 仪表板](https://dashboard.featbit.co/account)生成一份免费试用许可证密钥：

- 使用您的邮箱注册一个账户，或者通过 GitHub 登录。
- 在**账户**页面点击**生成试用许可证**。
  ![](./assets/request-trial-license1.png)
- 在弹出窗口中，输入您的公司名称和工作区 ID，然后点击**确定**
  ![](./assets/request-trial-license2.png)
- 在许可证表格中找到生成的许可证，然后点击“复制”。
  ![](./assets/request-trial-license3.png)

## 获取商业许可证密钥

如需商业许可证密钥，请发邮件至 [contact@featbit.co](mailto:contact@featbit.co) 或加入我们的 [Slack 频道](https://join.slack.com/t/featbit/shared_invite/zt-1ew5e2vbb-x6Apan1xZOaYMnFzqZkGNQ)，我们会尽快给您回复。
## 上传许可证密钥

现在，请返回到 [http://localhost:8081/en/workspace](http://localhost:8081/en/workspace)，进入**许可证设置**区域，输入您之前获取的许可证密钥，并点击**更新**。

![](./assets/save_license_key.png)

保存后，将鼠标悬停在“企业”按钮上，您将可以看到如下的许可证详细信息。

![](./assets/check-license.png)
