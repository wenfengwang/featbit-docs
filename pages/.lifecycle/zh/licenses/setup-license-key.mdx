# 设置许可证密钥

<Callout type="info">本文兼容 FeatBit 3.0.0 及以上版本。</Callout>

本指南说明了使用许可证密钥设置 FeatBit 的过程。

## 获取工作区 ID

在获取许可证密钥之前，需要工作区 ID。请按照下列步骤操作：

1. 确保您的 FeatBit 正在运行，并且我们假设 FeatBit UI 可以在 http://localhost:8081 访问
2. 使用您的电子邮件和密码登录 FeatBit。对于默认设置，请使用：
   - 用户名：test@featbit.com
   - 密码：123456
  
3. 访问 http://localhost:8081/en/workspace，在 **许可证设置** 部分，点击 **复制** 按钮
   
   ![](./assets/workspaceid.png)

## 获取免费试用许可证密钥

要评估所有 FeatBit 功能，请从 [FeatBit Dashboard](https://dashboard.featbit.co/account) 生成免费试用许可证密钥：

- 使用您的电子邮件创建账户或通过 GitHub 登录。
- 在 **账户** 页面点击 **生成试用许可证**。

  ![](./assets/request-trial-license1.png)
- 在弹出窗口中，输入您的公司名称和工作区 ID，然后点击 **确定**

  ![](./assets/request-trial-license2.png)
- 在许可证表格中找到生成的许可证，然后点击复制。

  ![](./assets/request-trial-license3.png)

## 获取商业许可证密钥

如需商业许可证密钥，请发送电子邮件至 [contact@featbit.co](mailto:contact@featbit.co) 或加入我们的 [Slack 频道](https://join.slack.com/t/featbit/shared_invite/zt-1ew5e2vbb-x6Apan1xZOaYMnFzqZkGNQ)，我们会及时回复。
## 上传许可证密钥

现在返回到 http://localhost:8081/en/workspace，进入 **许可证设置** 部分，输入之前获取的许可证密钥，然后点击 **更新**。

![](./assets/save_license_key.png)

保存后，将鼠标悬停在企业按钮上，您将能够查看许可证详细信息，如下图所示

![](./assets/check-license.png)
