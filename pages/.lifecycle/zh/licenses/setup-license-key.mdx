# 设置许可密钥

<Callout type="info">
  本文兼容 FeatBit 3.0.0 及以上版本。
</Callout>

本指南说明了如何使用许可密钥设置 FeatBit。

## 获取工作区 ID

在获取许可密钥之前，您需要工作区 ID。请按照以下步骤操作：

1. 确保 FeatBit 正在运行，我们假设 FeatBit UI 可在 http://localhost:8081 上访问
2. 使用您的电子邮件和密码登录 FeatBit。对于默认设置，请使用：
   - 用户名：test@featbit.com
   - 密码：123456
  
3. 转到 http://localhost:8081/en/workspace，在“**许可设置**”部分，单击“**复制**”按钮
   
   ![](./assets/workspaceid.png)

## 获取免费试用许可密钥

要评估所有 FeatBit 功能，您可以从 [FeatBit 仪表板](https://dashboard.featbit.co/account) 生成免费试用许可密钥：

- 使用您的电子邮件创建帐户或通过 GitHub 登录。
- 在“**帐户**”页面中，单击**生成试用许可证**。

  ![](./assets/request-trial-license1.png)
- 在弹出窗口中，输入您的公司名称和工作区 ID，然后单击**确定**

  ![](./assets/request-trial-license2.png)
- 找到生成的许可证并单击复制。

  ![](./assets/request-trial-license3.png)

## 获取商业许可密钥

如需商业许可密钥，请发送电子邮件至 [contact@featbit.co](mailto:contact@featbit.co) 或加入我们的 [Slack 频道](https://join.slack.com/t/featbit/shared_invite/zt-1ew5e2vbb-x6Apan1xZOaYMnFzqZkGNQ)，我们将尽快回复。
## 上传许可密钥

现在返回 http://localhost:8081/en/workspace，转到“**许可设置**”部分，输入之前获取的许可密钥，然后单击**更新**。

![](./assets/save_license_key.png)

保存后，将鼠标悬停在“企业”按钮上，您将能够查看许可详细信息，如下所示

![](./assets/check-license.png)
