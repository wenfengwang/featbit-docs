# 旗帜洞察

## 概览

本主题解释了如何利用**洞察**标签页上的功能标志评估图表，以观察功能标志随时间的表现。

<Callout type="info">
The insights data is near real time and may have some delay, if you see no data, just wait a while then refresh the page
</Callout>


## 使用功能标志评估图表

功能标志评估图表为您的工程和产品团队提供了洞察，让他们了解用户随时间接收到的每种功能标志变体的次数。

查看功能标志的评估图表，请按以下步骤操作：

1. 导航到功能标志列表。
2. 点击您想要查看其随时间评估的功能标志的名称。随后将显示该功能标志的**洞察**标签页。
3.  点击**洞察**标签页。此时将出现功能标志评估图表：

![](../feature-flags/assets/the-flag-insights/001.webp)

请注意，功能标志评估图表是基于每次对功能标志的评估生成的，而不是基于每个用户。这意味着，如果同一个用户进行了20次评估，功能标志评估图表将记录并展示全部20次评估。

## 使用最终用户的功能标志变体表

在功能标志评估图表下方，有一个最终用户的功能标志变体表，它向您的团队展示了最终用户随时间所接收到的不同变体。

![](../feature-flags/assets/the-flag-insights/002.webp)
