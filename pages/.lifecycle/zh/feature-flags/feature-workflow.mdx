# 功能工作流

### 概述 <a href="#overview" id="overview"></a>

本主题解释了如何通过在 FeatBit 中创建复杂的工作流来更精细地控制功能标志的使用。工作流映射到团队构建、发布和控制软件的方式。您还可以定义自定义发布策略，将 FeatBit 与组织现有的工具和流程进行集成。

### 可用功能 <a href="#available-features" id="available-features"></a>

要素工作流用户可使用以下功能：

* [标志触发器](flag-triggers)
* [计划的标志更改](scheduled-flag-changes)
