# 创建标志变体

### 概述 <a href="#overview" id="overview"></a>

本主题将解释如何创建和编辑功能标志的变体，并概述不同类型的标志变体。

### 了解标志类型 <a href="#understanding-flag-types" id="understanding-flag-types"></a>

FeatBit支持布尔和多变量标志。在**变体**部分，您可以添加、编辑或删除现有标志的变体：

* **布尔**标志有两种变体：`true`或`false`。
* **多变量**标志可以有两个以上的变体。允许的变体取决于标志的类型。多变量标志类型包括字符串、数字和JSON。

在**变体**部分，您可以更改变体类型，添加、编辑或删除现有标志的变体。

#### 删除变体

删除变体时，如果任何规则使用该变体，或者已将某些目标用户添加到该变体中，则会显示一条警告消息，您必须在删除变体之前修改这些规则。

### 更改默认标志值 <a href="#changing-default-flag-values" id="changing-default-flag-values"></a>

创建功能标志时，某些值被指定为默认值。标志的默认值是在标志打开或关闭时提供的值，除非您另行指定。当标志为**打开**时，默认值可以是单个变体或卷展。当标志为**关闭**时，默认值只能是单个变体。

例如，布尔标志在打开时可以将`true`设置为默认规则，关闭时将`false`设置为默认变体。多变量标志在打开时可以将`variation 2`设置为默认规则，关闭时将`variation 1`设置为默认变体，并配置`variation 3`和`variation 4`以在用户匹配某些定位规则时显示。

每次创建功能标志时，都会自动指定默认值。您可以接受默认值或更改它们。

您可以通过更改**如果关闭，则提供**或更改默认规则的提供值来更改标志的默认值。

![](../feature-flags/assets/create-flag-variations/002.webp)

### 了解多变量标志 <a href="#understanding-multivariate-flags" id="understanding-multivariate-flags"></a>

多变量标志允许您使用一个标志同时提供功能的两个以上变体。可以添加到多变量标志的变体数量没有限制，因此对于复杂的用例和管理功能的多个变体非常有用。

多变量标志变体类型包括字符串、数字或JSON。

您可以使用百分比卷展来优化功能的卷展。例如，将字符串作为变体的多变量标志可用于为恐龙游戏启用不同的难度，向用户提供这些类型的百分比推出。在此示例中，默认规则为10%的用户提供困难难度，10%的用户提供简单难度，80%的用户提供中等难度。

以下是带有百分比推出的标志的图像：

![](../feature-flags/assets/create-flag-variations/003.webp)

您可以使用自定义规则来定位多组用户并进一步优化卷展。在下一个示例中，第一条规则对年龄小于20岁的所有用户提供难度。第二条规则针对所在国家为FR或IT的用户，其中10%的用户具有难度，10%的用户具有简单难度，80%的用户具有中等难度。所有未在自定义规则中作为目标的用户都将使用默认规则。

以下是具有自定义规则的标志的图像：

![](../feature-flags/assets/create-flag-variations/004.webp)

您还可以使用多变量标志执行FeatBit的实验功能。要了解更多信息，请阅读[创建实验](../experimentation/creating-experiments.md)。
