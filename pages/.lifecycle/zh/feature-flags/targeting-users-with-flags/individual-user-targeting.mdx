# 个别用户定位

## 概述 <a href="#overview" id="overview"></a>

本主题将解释如何使用标志的**定位**选项卡来控制哪些个别用户会收到功能标志的变体。

## 将用户分配到不同变体 <a href="#assigning-users-to-a-variation" id="assigning-users-to-a-variation"></a>

在**定位**选项卡的“目标用户”部分，您可以将单个用户分配到特定的标志变体。

> 仅对少量用户进行个别定位
>
> 我们建议仅对极少量的个别用户进行个别定位。每个变体个别定位超过 100 个用户可能会导致性能下降。

如果您的应用程序已经将数据发送回 FeatBit，您可以按名称或用户密钥搜索用户。这些字符串区分大小写。将鼠标悬停在用户行上，可以查看该用户的更多属性。

如果您需要定位 FeatBit 不知道的用户，您可以手动输入他们的密钥。这些用户将以黄色显示，直到遇到功能标志。

在下面的屏幕截图中，特定用户正在接收标志的第一个变体。该标志是针对他们的。这意味着这些用户在访问该标志时将收到该标志的变体。

## 移除已定位的用户 <a href="#removing-targeted-users" id="removing-targeted-users"></a>

您可以通过单击用户的名称来将其从标志的定位列表中移除。操作步骤如下：

1. 转到标志的**定位**选项卡。
2. 在“目标用户”部分，找到要移除的用户的名称。
3. 单击其名称旁边的 **X** 以将其移除：
