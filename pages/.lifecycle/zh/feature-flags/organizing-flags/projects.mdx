# 项目

## 概述

本主题介绍了在FeatBit中的项目是什么，以及您如何使用它们来管理组织中的不同团队和任务。

## 了解项目

项目让您可以从一个FeatBit账户管理多个不同的业务目标。例如，您可以创建一个名为“电子商务”的项目和另一个名为“会计”的项目。每个项目都有其独特的环境集和特性标志。

环境是包含在项目中的组织单元。要了解更多，请阅读[环境](environments.md)。

一个常见的模式是将您的FeatBit实例中的每个项目映射到公司制作的特定产品。例如，一个移动应用、Web应用和后端服务可能都属于一个项目，因为它们作为一个产品呈现给用户时，很可能会共享大部分特性标志。

## 项目和环境

您可以在每个项目中创建多个环境。所有项目至少必须有一个环境。

## 创建新项目

要创建新项目，请执行以下操作：

1. 导航至**组织**页面。
2. 点击**项目**标签。
3.  点击**添加**。将出现“添加项目”抽屉界面：&#x20;

![](../../feature-flags/assets/organizing-flags/projects/002.webp)

4. 为您的项目起一个易于理解的名称。
5. 点击**保存**。

您已成功创建了一个项目。它会出现在列表中。

## 删除项目

当您不再需要某个项目时，您可以删除它。

<Callout type="warning">请谨慎删除，因为删除项目是不可逆的，并且会连同所有环境和特性标志一起删除。</Callout>

要删除项目：

1. 导航至**组织**页面。
2. 点击**项目**标签。
3. 找到您想要删除的项目。
4.  点击**移除**。将出现一个确认对话框。

![](../../feature-flags/assets/organizing-flags/projects/003.png)

5. 点击**确定**。

您的项目现在已被删除。
