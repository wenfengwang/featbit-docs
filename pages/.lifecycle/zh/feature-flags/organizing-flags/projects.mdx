# 项目

## 概述

本主题将解释 FeatBit 中的项目是什么，以及如何使用它们来管理组织中的不同团队和任务。

## 了解项目

项目允许您从一个 FeatBit 帐户管理多个不同的业务目标。例如，您可以创建一个名为“电子商务”的项目和另一个名为“会计”的项目。每个项目都有自己独特的环境和功能标志集。

环境是项目中包含的组织单位。要了解更多信息，请阅读 [环境](environments.md)。

一个常见的模式是将 FeatBit 实例中的每个项目映射到公司生产的特定产品。例如，作为产品呈现给用户的移动应用程序、Web 应用和后端服务可能都适合一个项目，因为它们可能会共享大多数标志。

## 项目和环境

您可以在每个项目中创建多个环境。所有项目必须至少有一个环境。

## 创建新项目

要创建新项目：

1. 导航到 **组织** 页面。
2. 点击 **项目** 选项卡。
3. 点击 **添加**。将出现“添加项目”抽屉：&#x20;

![](../../feature-flags/assets/organizing-flags/projects/002.webp)

4. 为您的项目指定一个易于理解的名称。
5. 点击 **保存**。

您刚刚创建了一个项目，它将显示在列表中。

## 删除项目

当您不再使用某个项目时，可以将其删除。

<提示框 type="warning">
请谨慎删除，因为删除项目是永久性的，也会删除其中的所有环境和标志。
</提示框>

要删除项目：

1. 导航到 **组织** 页面。
2. 点击 **项目** 选项卡。
3. 找到要删除的项目。
4.  点击 **删除**。将显示一个确认对话框。

![](../../feature-flags/assets/organizing-flags/projects/003.png)

5. 点击 **确定**。

您的项目现已删除。
