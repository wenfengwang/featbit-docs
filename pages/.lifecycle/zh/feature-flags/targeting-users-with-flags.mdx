# 针对用户使用标志

## 概述 <a href="#overview" id="overview"></a>

本类别解释了如何使用标志的**定位**选项卡来控制哪些用户会收到功能标志的变体。配置哪些用户接收哪个标志变体被称为标志的定位。

## 使用定位选项卡 <a href="#using-the-targeting-tab" id="using-the-targeting-tab"></a>

您可以使用**定位**选项卡在执行更广泛的部署之前推出功能，进行内部测试、私人测试版或可用性测试。如果您知道您只希望用户在特定时间段内收到标记（未来功能），甚至可以设置标记定位的到期日期。

虽然**定位**选项卡使用了“用户”作为目标的语言，但用户可以是任何唯一对应目标的标识符。您可以定位您的应用程序用户、电子邮件地址、系统、服务、计算机、资源或任何其他可以唯一标识的内容。
