# 针对用户使用特性标志

## 概览 <a href="#overview" id="overview"></a>

本节介绍如何使用特性标志的**定位**标签来控制哪些用户接收特性标志的不同变体。配置哪些用户接收哪种变体被称为特性标志的定位。

## 使用定位标签 <a href="#using-the-targeting-tab" id="using-the-targeting-tab"></a>

您可以使用**定位**标签为内部测试、私人测试版或可用性测试推出特性，在进行更广泛的推广之前。如果您已知只想在特定时间段内让用户接收特性标志，您甚至可以设置特性标志定位的过期日期（未来功能）。

尽管**定位**标签使用了针对“用户”的表述，但“用户”可以是任何与目标唯一对应的标识符。您可以针对您的应用程序用户、电子邮件地址、系统、服务、计算机、资源或任何其他可以唯一识别的对象进行定位。
