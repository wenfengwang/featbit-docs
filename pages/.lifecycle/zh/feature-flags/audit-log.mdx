# 审计日志

### 概览 <a href="#overview" id="overview"></a>

本主题解释了如何使用审计日志来跟踪 FeatBit 中所做的更改。

### 使用审计日志 <a href="#using-the-audit-log" id="using-the-audit-log"></a>

**审计日志** 是对给定环境中功能标志所做更改的运行计数。您可以从侧边菜单访问审计日志。

审计日志仅显示当前环境中功能标志的更改。要查看其他环境的更改，请使用右上角的 **切换环境** 菜单。

默认情况下，审计日志显示最近的十个条目。您可以使用日期选择器和其他筛选器来筛选审计日志显示的条目。
