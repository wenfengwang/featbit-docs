# 最终用户

## 概述

本主题介绍了如何下载和上传当前环境的最终用户。

以下是下载/上传最终用户页面的图片：

![](../data-import-and-export/assets/end-users/001.webp)

## 下载

要下载当前环境的最终用户：

1. 导航到 **数据同步** 页面。
2. 点击 **下载/上传** 选项卡。
3. 点击 **下载**。

服务器将返回一个包含所有最终用户和所有用户属性的json文件。

## 上传

要上传当前环境的最终用户：

1. 导航到 **数据同步** 页面。
2. 点击 **下载/上传** 选项卡。
3. 点击 **上传**。上传最终用户抽屉将出现：

![](../data-import-and-export/assets/end-users/002.webp)

<提示框 type="info">
当您将最终用户上传到服务器时：

- 对于用户，只会执行插入/更新操作。
- 对于用户属性，只会执行插入操作。
</提示框>
