# 末端用户

## 概览

本主题解释了如何下载和上传当前环境的末端用户。

以下是下载/上传末端用户页面的图片：

![](../data-import-and-export/assets/end-users/001.webp)

## 下载

要下载当前环境的末端用户，请按照以下步骤操作：

1. 导航至**数据同步**页面。
2. 点击**Download/Upload**标签页。
3. 点击**下载**按钮。

服务器随后会返回一个包含所有末端用户及其属性的 json 文件。

## 上传

要上传当前环境的末端用户，请按照以下步骤操作：

1. 导航至**数据同步**页面。
2. 点击**下载/上传**标签页。
3.  点击 **上传** 按钮。此时会出现上传末端用户的侧边栏：

![](../data-import-and-export/assets/end-users/002.webp)

<Callout type="info">
When you upload end-users to the server:


- 对于用户，系统将只执行插入/更新操作。
- 对于用户属性，系统将只执行插入操作。
