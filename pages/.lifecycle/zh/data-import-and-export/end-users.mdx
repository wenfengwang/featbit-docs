# 终端用户

## 概述

本主题解释如何下载和上传当前环境的终端用户。

这是下载/上传终端用户页面的图片：

![](../data-import-and-export/assets/end-users/001.webp)

## 下载

下载当前环境的终端用户：

1. 导航至 **数据同步** 页面。
2. 点击 **下载/上传** 标签页。
3. 点击 **下载**。

服务器随后会返回一个包含所有终端用户及其属性的 json 文件。

## 上传

上传当前环境的终端用户：

1. 导航至 **数据同步** 页面。
2. 点击 **下载/上传** 标签页。
3. 点击 **上传**。此时会出现上传终端用户的侧边栏：

![](../data-import-and-export/assets/end-users/002.webp)

<Callout type="info">
当您向服务器上传终端用户时：

- 对于用户，只会执行插入/更新操作。
- 对于用户属性，只会执行插入操作。
</Callout>
