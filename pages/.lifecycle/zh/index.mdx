
# 概览

FeatBit 是一个快速、可扩展并且广受欢迎的开源特性标志管理服务。

- **安全发布代码**。通过生产测试来减轻风险，最初只向1%的用户推出新特性，然后逐步扩大范围，并确保在无需重新部署的情况下即时修复错误。
- **加速创新**。将特性的部署与产品发布分离，以尽量减少合并时的冲突。随心所欲地部署，在老板的请求下即刻发布任何特性。
- **定向体验**。向特定的目标用户发布特性，并持续评估和提升产品的商业价值，同时减少对开发者的依赖。
- **为开发者设计**。使用简单的if/else语句来控制和发布特性，消除复杂的DevOps任务。这使开发者能够直接推动业务价值。
