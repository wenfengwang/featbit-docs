# 概要

FeatBitは、高速でスケーラブル、かつ人気の高いオープンソースのフィーチャーフラグ管理サービスです。

- **安全にコードをリリース**。本番環境でのテストによりリスクを軽減し、新機能を最初に1%のユーザーにロールアウトしてから拡大し、再デプロイなしで即時にエラー回復が可能です。
- **より速くイノベーションを**。機能のデプロイをリリースから分離して、マージコンフリクトを最小限に抑えます。自由にデプロイし、上司の要請に応じて任意の機能を即座にリリースできます。
- **ターゲット指定されたエクスペリエンス**。特定のターゲットユーザーに機能をリリースし、製品のビジネス価値を継続的に測定・改善する一方で、開発者の介入を減らします。
- **開発者のために誕生**。シンプルなif/else文を使用して機能をコントロールし、リリースすることで、複雑なDevOpsタスクを排除します。これにより開発者は直接ビジネス価値を生み出すことができます。
