# 3. SDKの接続
SDKの使用は、ソフトウェアをフィーチャーフラグを介して提供、制御、実験する方法を理解するための不可欠なスキルです。

最後のステップ「デモとのやり取りを試す」で、ゲームコンポーネントを表示/非表示にしたり、ゲームの難易度モードを切り替える方法を学びました。次は、コードでの実装方法を見てみましょう。

## ポータル内のガイドに従う
最も簡単な方法は、ポータルの「はじめに」ページの手順に従うことです。ポータルでは、次のことができます

**ステップ 1**. 前のステップで作成したフィーチャーフラグの1つを選択します。次に、「次へ」ボタンをクリックしてステップ2に進みます。

![](../getting-started/assets/connect-to-sdk/001.png)

**ステップ 2**. 自分の使い慣れたプログラミング言語を選択します。自動的に生成されたコードをコピーしてIDEで実行します。

![](../getting-started/assets/connect-to-sdk/002.png)

ポータルによって生成されたコードには、次のようなすべての必要な要素が含まれています。

- SDKのインストール
- サーバーURL、環境キーを使用したSDKの初期化
- カスタマイズされたプロパティを持つユーザーの識別
- フィーチャーフラグの実装
- フィーチャーフラグの評価
- フィーチャーフラグの更新リスナー
- その他

![](../getting-started/assets/connect-to-sdk/003.png)


## SDKを手動で接続する

クライアントサイド、サーバーサイド、モバイルのSDKを幅広く提供しています。自分の得意な言語を選んでクイックスタートを完了するのに役立ちます。

<Callout type="info" emoji="👾">
  私たちのSDKを使用することをお勧めします。なぜなら、私たちがすでにベストプラクティスを実装しているからです。
</Callout>

### Javascript

クライアントサイドの[Javascript/Typescript SDK](https://github.com/featbit/featbit-js-client-sdk)を提供しています。また、[React](https://github.com/featbit/featbit-samples/tree/main/samples/dino-game/interactive-demo-react)および[Vue](https://github.com/featbit/featbit-samples/tree/main/samples/dino-game/interactive-demo-vue)向けの「Dino Game」サンプルも提供しています。このSDKは、Typescript、React、Vue、Angularなどと完全に互換性があります。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれています。また、お試しいただけるサンプルプロジェクトも提供しています。

### .NET

[サーバーサイドの.NET SDK](https://github.com/featbit/featbit-dotnet-sdk)も提供しています。このSDKは、コンソールアプリやASP.NET Coreアプリで使用できます。

また、WPF、WinForm、MAUIなどのフロントエンドアプリケーションにも対応しています。ただし、実際のモバイルSDKよりもバッテリー消費が大きいため、モバイルアプリケーションでは使用をお勧めしません。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれています。また、お試しいただけるサンプルプロジェクトも提供しています。

### Java

[Java SDK](https://github.com/featbit/featbit-java-sdk)も提供しています。このSDKは、コンソールアプリやSpring Bootアプリで使用できます。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれています。また、お試しいただけるサンプルプロジェクトも提供しています。

### Python

[Python SDK](https://github.com/featbit/featbit-python-sdk)も提供しています。SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれています。また、お試しいただけるサンプルプロジェクトも提供しています。

### Go

[Go SDK](https://github.com/featbit/featbit-go-sdk)も提供しています。SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれています。また、お試しいただけるサンプルプロジェクトも提供しています。


### REST APIとWebSocket

まだSDKを提供していないその他の言語やフレームワークの場合は、REST APIを使用してフィーチャーフラグを取得できます。ドキュメントは[こちら](../api-docs/get-users-feature-flags-with-api)を参照してください。

リアルタイムのフィーチャーフラグ更新を取得するには、WebSocketインターフェースをご利用いただけます。この機能についての詳細情報をお知りになるには、私たちのSlackチャンネルにご参加ください。
