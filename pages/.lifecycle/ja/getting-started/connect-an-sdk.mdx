# 3. SDK を接続する

SDK の使用は、フィーチャーフラグを通じてソフトウェアの提供、制御、および実験を行う方法を理解する上で不可欠なスキルです。

最後のステップ「デモとの対話を試す」では、フィーチャーフラグを使用してゲームコンポーネントの表示/非表示を切り替えたり、ゲームの難易度モードを変更する方法を学びました。それでは、コード内でどのように実装されているか見てみましょう。

## ポータルのガイドに従う

最も簡単な方法は、ポータルの「Getting Started」ページの指示に従うことです。ポータルでは、以下の操作が可能です。

**ステップ1**。前のステップで作成したフィーチャーフラグの1つを選択します。次に、「次へ」ボタンをクリックしてステップ2に進みます。

![](../getting-started/assets/connect-to-sdk/001.png)

**ステップ2**。あなたが使い慣れたプログラミング言語を選択してください。自動生成されたコードをコピーし、IDEで実行します。

![](../getting-started/assets/connect-to-sdk/002.png)

ポータルによって生成されたコードには、以下のような必要なすべての要素が含まれています。

- SDKのインストール
- サーバーURL、環境キーを用いたSDKの初期化
- カスタマイズされたプロパティを持つユーザーの識別
- フィーチャーフラグの実装
- フィーチャーフラグの評価 
- フィーチャーフラグ更新リスナー
- その他

![](../getting-started/assets/connect-to-sdk/003.png)


## SDK を手動で接続する

クライアントサイド、サーバーサイド、モバイル向けの様々なSDKを提供しています。得意な言語を選択して、クイックスタートを迅速に完了させましょう。

<Callout type="info" emoji="👾">
  私たちのSDKを使用することをお勧めします。なぜなら、私たちは既にベストプラクティスを実装しているからです。
</Callout>

### JavaScript

クライアントサイドの [JavaScript/TypeScript SDK](https://github.com/featbit/featbit-js-client-sdk) を提供しています。また、[React](https://github.com/featbit/featbit-samples/tree/main/samples/dino-game/interactive-demo-react) および [Vue](https://github.com/featbit/featbit-samples/tree/main/samples/dino-game/interactive-demo-vue) 用の「Dino Game」サンプルも提供しています。このSDKは、TypeScript、React、Vue、Angularなどで完璧に動作します。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれており、試用できるサンプルプロジェクトも提供されています。

### .NET

サーバーサイドの [.NET SDK](https://github.com/featbit/featbit-dotnet-sdk) を提供しています。このSDKは、コンソールアプリケーションやASP.NET Coreアプリケーションで使用できます。

WPF、WinForms、MAUIなどのフロントエンドアプリケーションでも動作しますが、実際のモバイルSDKよりもバッテリー消費が多いため、モバイルアプリケーションでの使用は推奨しません。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれており、試用できるサンプルプロジェクトも提供されています。

### Java

[Java SDK](https://github.com/featbit/featbit-java-sdk) を提供しています。このSDKは、コンソールアプリケーションやSpring Bootアプリケーションで使用できます。

SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれており、試用できるサンプルプロジェクトも提供されています。

### Python

[Python SDK](https://github.com/featbit/featbit-python-sdk) を提供しています。SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれており、試用できるサンプルプロジェクトも提供されています。

### Go

[Go SDK](https://github.com/featbit/featbit-go-sdk) を提供しています。SDKのGitHubリポジトリには、その使用方法に関する完全なチュートリアルが含まれており、試用できるサンプルプロジェクトも提供されています。


### REST API と WebSocket

まだSDKを提供していない他の言語やフレームワークについては、REST APIを使用してフィーチャーフラグを取得できます。ドキュメントは[こちら](../api-docs/get-users-feature-flags-with-api)を参照してください。

WebSocketインターフェースを使用してリアルタイムにフィーチャーフラグの更新を取得することもできます。この機能についての詳細は、私たちのSlackチャンネルに参加して情報を得てください。
