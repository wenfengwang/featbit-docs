# エンドユーザー

## 概要

このトピックでは、現在の環境にエンドユーザーをダウンロードおよびアップロードする方法について説明します。

以下は、エンドユーザーのダウンロード/アップロードページのイメージです:

![](../data-import-and-export/assets/end-users/001.webp)

## ダウンロード

現在の環境のエンドユーザーをダウンロードするには、以下の手順を実行してください:

1. **データ同期** ページに移動します。
2. **ダウンロード/アップロード** タブをクリックします。
3. **ダウンロード** をクリックします。

サーバーからは、すべてのエンドユーザーとすべてのユーザープロパティが含まれた json ファイルが返されます。

## アップロード

現在の環境のエンドユーザーをアップロードするには、以下の手順を実行してください:

1. **データ同期** ページに移動します。
2. **ダウンロード/アップロード** タブをクリックします。
3. **アップロード** をクリックします。アップロードエンドユーザーのドロワーが表示されます:

![](../data-import-and-export/assets/end-users/002.webp)

<Callout type="info">
エンドユーザーをサーバーにアップロードする場合:

- ユーザーに対しては、挿入/更新操作のみが実行されます。
- ユーザープロパティに対しては、挿入操作のみが実行されます。
</Callout>
