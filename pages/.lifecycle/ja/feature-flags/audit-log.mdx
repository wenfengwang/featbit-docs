# 監査ログ

### 概要 <a href="#overview" id="overview"></a>

このトピックでは、FeatBitで行われた変更を追跡するために監査ログを使用する方法について説明します。

### 監査ログの使用方法 <a href="#using-the-audit-log" id="using-the-audit-log"></a>

**監査ログ**は、特定の環境でのフィーチャーフラグの変更履歴の一覧です。サイドメニューから監査ログにアクセスできます。

監査ログでは、現在の環境におけるフィーチャーフラグの変更を表示します。他の環境の変更を表示するには、右上の**環境の切り替え**メニューを使用します。

デフォルトでは、監査ログには最新の10件のエントリが表示されます。日付ピッカーや他のフィルタを使用して、監査ログに表示されるエントリを絞り込むことができます。
