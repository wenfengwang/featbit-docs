# 監査ログ

### 概要 <a href="#overview" id="overview"></a>

このトピックでは、FeatBitで変更を追跡するための監査ログの使用方法について説明します。

### 監査ログの使用 <a href="#using-the-audit-log" id="using-the-audit-log"></a>

**監査ログ**は、特定の環境で機能フラグに対して行われた変更の実行記録です。サイドメニューから監査ログにアクセスできます。

監査ログは、現在の環境に対する機能フラグの変更のみを表示します。他の環境の変更を表示するには、右上にある**環境の切り替え**メニューを使用してください。

デフォルトでは、監査ログは最新の10件のエントリを表示します。日付ピッカーやその他のフィルターを使用して、監査ログが表示するエントリをフィルタリングすることができます。
