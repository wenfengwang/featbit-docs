# 監査ログ

### 概要 <a href="#overview" id="overview"></a>

このトピックでは、FeatBitで行われた変更を追跡するための監査ログの使用方法について説明します。

### 監査ログの使用方法 <a href="#using-the-audit-log" id="using-the-audit-log"></a>

**監査ログ**は、特定の環境内でfeature flagsに加えられた変更の連続した記録です。サイドメニューから監査ログにアクセスできます。

監査ログは、現在の環境におけるfeature flagsの変更のみを表示します。他の環境での変更を見るには、右上にある**環境切替**メニューを使用してください。

デフォルトでは、監査ログには最新の10件のエントリが表示されます。日付ピッカーやその他のフィルターを使用して、監査ログに表示されるエントリをフィルタリングすることができます。
