# 機能ワークフロー

### 概要 <a href="#overview" id="overview"></a>

このトピックでは、FeatBit内で複雑なワークフローを作成することで、機能フラグの使用をより細かく制御する方法について説明します。ワークフローは、チームがソフトウェアを構築、出荷、および制御する方法に対応します。また、FeatBitを組織の既存のツールやプロセスと統合するカスタムリリース戦略を定義することもできます。

### 利用可能な機能 <a href="#available-features" id="available-features"></a>

機能ワークフローのユーザーが利用できる以下の機能があります：

* [フラグトリガー](flag-triggers)
* [スケジュールされたフラグの変更](scheduled-flag-changes)
