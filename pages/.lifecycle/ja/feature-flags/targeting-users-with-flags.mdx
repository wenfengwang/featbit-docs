# フラグを使用してユーザーをターゲットにする

## 概要 <a href="#overview" id="overview"></a>

このカテゴリでは、**ターゲット**タブを使用して、フィーチャーフラグのバリエーションを受け取るユーザーを制御する方法について説明します。どのユーザーがどのフラグのバリエーションを受け取るかを設定することは、フラグのターゲティングと呼ばれます。

## ターゲットタブの使用 <a href="#using-the-targeting-tab" id="using-the-targeting-tab"></a>

**ターゲット**タブを使用して、機能を内部テスト、プライベートベータ版、または使用性テストに展開し、より広範な展開を行う前に試運転することができます。また、フラグの対象ターゲティングに有効期限を設定することもできます。たとえば、フラグを特定の期間だけ受け取ることを希望する場合（将来の機能）。

**ターゲット**タブでは、対象となる「ユーザー」という言葉が使用されますが、ユーザーとはターゲットに一意に対応する識別子であることができます。アプリケーションのユーザー、メールアドレス、システム、サービス、マシン、リソースなど、一意に識別できるものすべてをターゲットにすることができます。
