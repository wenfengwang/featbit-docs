# ユーザーとユーザーセグメント

このカテゴリーには、FeatBitのユーザーに関するドキュメントトピックがあります。ユーザーとは、製品で機能フラグに遭遇する人、サービス、マシン、または他のリソースのことです。

ターゲットに一意に対応する任意の識別子がユーザーとして指定されます。FeatBit SDKによって識別された後、**ユーザー**リストに表示されます。ユーザーが機能フラグとどのように互動するかを制御し、どのユーザーがどのフラグに遭遇するかを、ユーザー属性やユーザーセグメントにフラグをターゲティングすることで管理することができます。収集するユーザーデータに含まれるユーザーを選択したり除外したりすることもできます。

機能フラグとのやりとりにおいて、ユーザーとユーザーセグメントがどのように相互作用するかを理解するために、このカテゴリーのコンテンツを読んでください。
