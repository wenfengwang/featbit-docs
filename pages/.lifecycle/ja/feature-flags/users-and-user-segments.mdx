# ユーザーとユーザーセグメント

このカテゴリには、FeatBitのユーザーに関するドキュメントトピックスがあります。ユーザーは、製品内でフィーチャーフラグに遭遇する人、サービス、機械、またはその他のリソースです。

ターゲットに対応する一意の識別子は、ユーザーとして指定されます。FeatBit SDKによって識別された後、**ユーザー**リストに表示されます。ユーザー属性やユーザーセグメントにフラグをターゲット指定することで、ユーザーがフラグとどのように相互作用するか、どのユーザーがどのフラグに遭遇するかを管理できます。さらに、収集するユーザーデータに含めるか除外するか、どのユーザーを選択するかを選択することもできます。

このカテゴリのコンテンツを読んで、ユーザーとユーザーセグメントがフィーチャーフラグとどのように相互作用するかを理解してください。
