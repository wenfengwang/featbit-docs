# ユーザーとユーザーセグメント

このカテゴリには、FeatBitのユーザーに関するドキュメントトピックが含まれています。ユーザーとは、製品内でフィーチャーフラグに遭遇する人々、サービス、マシン、またはその他のリソースを指します。

ターゲットに一意に対応する識別子はユーザーとして指定されます。これらは、FeatBit SDKによって識別された後に**ユーザー**リストに表示されます。ユーザー属性やユーザーセグメントにフィーチャーフラグをターゲティングすることで、ユーザーがフィーチャーフラグとどのように対話するか、どのユーザーがどのフラグに遭遇するかを管理できます。また、収集するユーザーデータに含めるか除外するユーザーを選択することもできます。

このカテゴリの内容を読むことで、ユーザーとユーザーセグメントがフィーチャーフラグとどのようにインタラクトするかを理解できます。
