# 审计日志

### 概述 <a href="#overview" id="overview"></a>

本主题介绍如何使用审计日志来跟踪在FeatBit中所做的更改。

### 使用审计日志 <a href="#using-the-audit-log" id="using-the-audit-log"></a>

**审计日志**是对给定环境中的特性标志所做更改的持续更新。您可以从侧边菜单访问审计日志。

审计日志仅显示当前环境中特性标志的更改。要查看其他环境的更改，请使用右上角的**切换环境**菜单。

默认情况下，审计日志显示最近的十个条目。您可以使用日期选择器和其他过滤器来筛选出审计日志显示的条目。
