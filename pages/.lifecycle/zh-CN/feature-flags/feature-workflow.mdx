# 功能工作流程

### 概述 <a href="#overview" id="overview"></a>

本主题解释了如何通过在 FeatBit 中创建复杂的工作流来更精细地控制功能标志的使用。工作流映射到您的团队构建、发布和控制软件的方式。您还可以定义自定义发布策略，将 FeatBit 与组织的现有工具和流程集成。
