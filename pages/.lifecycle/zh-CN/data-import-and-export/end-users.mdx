# 终端用户

## 概述

本主题介绍如何下载和上传当前环境的终端用户。

以下是下载/上传终端用户页面的示例图片：

![](../data-import-and-export/assets/end-users/001.webp)

## 下载

要下载当前环境的终端用户，请按照以下步骤操作：

1. 导航到**数据同步**页面。
2. 单击**下载/上传**选项卡。
3. 单击**下载**。

服务器将返回一个包含所有终端用户和所有用户属性的json文件。

## 上传

要上传当前环境的终端用户，请按照以下步骤操作：

1. 导航到**数据同步**页面。
2. 单击**下载/上传**选项卡。
3. 单击**上传**。上传终端用户抽屉将出现：

![](../data-import-and-export/assets/end-users/002.webp)

<Callout type="info">
当您上传终端用户到服务器时：

- 对于用户，只会执行插入/更新操作。
- 对于用户属性，只会执行插入操作。
</Callout>
